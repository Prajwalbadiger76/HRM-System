Project Name: HRM Management System
Technology Stack: MERN (MongoDB, Express.js, React.js, Node.js) + Java Spring Boot for Payroll Microservice
Database: MongoDB
Authentication: JWT Based Authentication

Description:
HRM Management System is a full-stack web application designed to manage HR operations like employee records, attendance tracking, leave management, payroll calculation, finance dashboard, and backup functionality. The system is created to be lightweight, fast, easy to use, and structured like real corporate HRMS products.

The application contains an admin dashboard, employee pages, forms, lists, API integrations, database interaction, file uploads, automation scripts, and documentation.

Project Objective:
To build a production-like HRM system, demonstrate integration between MERN stack and Java microservice, authentication, CRUD operations, optimized DSA logic, and database backup and restore automation.
